# 页面结构

ClaudeMaster 使用 hash 路由，所有页面通过 `#/path` 访问。

## 工作台（#/dashboard）

首页。展示待审批、工作中、待命中、最近会话四个区块。支持新建会话。

详见 [工作台](dashboard) 文档。

## 会话历史（#/sessions）

跨项目浏览所有历史对话。支持：
- 按项目筛选
- 关键词搜索（全文检索，300ms 防抖）
- 日期范围筛选（全部 / 今天 / 7天 / 30天）
- 分页浏览（每页 40 条，滚动加载更多）
- 新建会话

## 对话查看器（#/viewer/:project/:sessionId）

阅读完整对话内容，也是实时交互的主界面。支持：
- Markdown 渲染（用户和助手消息）
- 工具调用折叠展示（名称、输入、输出）
- 思维过程折叠展示
- Diff 代码高亮
- 实时聊天（连接 broker 管理的会话时）
- 权限审批对话框
- 消息批量加载（每次 30 条）

## 机器管理（#/agents）

管理本地和远程 cm-agent 连接。支持：
- 查看所有 agent 的在线状态、延迟、版本
- 查看远程 agent 上运行的 Claude Code 进程
- 远程进程批量终止
- Agent 显示名称编辑
- 添加新机器引导（显示安装命令）
- 在远程 agent 上新建会话

## 设置（#/settings）

- 深色/浅色主题切换
- 访问令牌配置（局域网认证）

## 产品文档（#/docs）

本文档。在平台内实时浏览，作为开发参考。支持多页导航和页内目录。
